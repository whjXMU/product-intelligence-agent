# Agent MVP 子窗口参考资料

本目录保存子窗口围绕 `packages/agent-mvp` 生成的参考文档。

## 主窗口定性

这些文档是 spike 后的思考材料，不是当前正式路线的最高优先级来源。

正式路线以以下文件为准：

- `agent_share_doc/project-status.md`
- `agent_share_doc/handoff.md`
- `docs/roadmap.md`
- `docs/architecture.md`
- `docs/adr/0002-agent-mvp-spike-boundary.md`

## 使用方式

可以参考：

- Agent 能力拆分思路；
- trace、artifact、eval、model gateway 等概念；
- 从实验包迁移到正式系统时的风险提示。

不要直接照搬：

- 阶段顺序；
- 包边界；
- 实现清单；
- “以本文为准”式的架构结论。

正式执行任何 Agent 工程化任务前，必须先回到主窗口确认当前阶段是否已经进入正式 Agent 实现。
